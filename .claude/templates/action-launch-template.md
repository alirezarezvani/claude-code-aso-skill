## Common Launch Issues

### Issue: App Store Rejection (Apple)

**Common Reasons:**
1. Metadata doesn't match app functionality
2. Missing demo account or functionality
3. Privacy policy issues
4. Guideline violations

**Solution:**
1. Review rejection reason carefully
2. Consult `submission-guide.md` for detailed fixes
3. Make required changes
4. Respond to reviewer with explanation
5. Resubmit

**Timeline:** Additional 24-48 hours for re-review

### Issue: Play Store Rejection (Google)

**Common Reasons:**
1. Content rating inaccurate
2. Policy violations (misleading claims)
3. Technical issues (crashes on testing)

**Solution:**
1. Review policy center for specific violation
2. Consult `submission-guide.md` for fixes
3. Make required changes
4. Resubmit

**Timeline:** Additional 2-7 days for re-review

### Issue: Low Initial Conversion Rate

**Potential Causes:**
- Screenshots don't communicate value
- Title/subtitle unclear
- Poor visual assets

**Solution:**
1. Review A/B test insights from Phase 3
2. Iterate on visual assets quickly
3. Test alternative messaging
4. Monitor for improvements
